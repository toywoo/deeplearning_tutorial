## Deep Learning Basic
### Image Classification: a core task in computer vision
어떤 이미지를 보여주고 그 이미지가 어떤 것인지 맞추는 것이다.
어떻게 할까 일반적인 예로 선형 분류가 있다. 선형 분류기는 데이터를 분리 시켜준다. 하지만 복잡한 데이터를 분리하기는 어렵다 그래서 다양하고 적합한 알고리즘을 사용한다. 신경망을 사용한다. 모델링을 쌓아서 비선형문제를 해결할 수 있게 된다.

### Perceiving and Understanding the Vision World

Classification, Semantic Segmetation, Object Detection, Instance Segmentation
