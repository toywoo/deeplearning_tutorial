# CS231n Lecture Summaries

This directory contains synthesized summaries of the CS231n: Deep Learning for Computer Vision lecture materials.

## Content Overview
- **`lec1.md`**: Introduction to Computer Vision and history.
- **`lec2.md` & `lec3.md`**: Linear classification, loss functions (SVM, Softmax), and optimization.
- **`lec5.md` - `lec7.md`**: Neural Networks, backpropagation, and Convolutional Neural Networks (CNNs).

## Writing Standards
1.  **Technical Terms**: Always provide clear Korean/English definitions for new terms (e.g., "Backpropagation (오차 역전파)").
2.  **Mathematical Clarity**: Use LaTeX for formulas where appropriate.
3.  **Visual References**: Reference images in `../../images/` using relative paths.
4.  **Consistency**: Maintain the structure of summarizing the "Core Question" -> "Key Mechanism" -> "Mathematical Foundation".

## Usage
These files are intended as study aids and reference points for the implementations found in the `source/` directories of this project.
