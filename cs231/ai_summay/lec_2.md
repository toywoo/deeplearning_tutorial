# Lecture 2: Image Classification with Linear Classifiers

## 1. 단계별 요약 (Step-by-step Summary)
- **Step 1: 컴퓨터 비전의 고질적 난제(Semantic Gap)**: 인간이 인지하는 고양이(고유의 의미)와 컴퓨터가 보는 0~255 숫자 텐서의 괴리를 극복하는 첫 이론인 데이터 중심 접근법(Data-driven Approach)을 소개합니다.
- **Step 2: K-최근접 이웃(k-NN)**: 단순히 입력 이미지와 훈련 데이터 사이의 거리 오차(L1, L2 거리)를 계산해 가장 비슷한 것을 라벨로 삼는 알고리즘. 하지만 속도가 끔찍하게 느리며(테스트타임 극악) 의미적 차이를 잡지 못한다는 약점을 짚습니다. 
- **Step 3: 선형 분류기(Linear Classifiers)**: 모델 파라미터(행렬 W)를 학습하여 입력 사진 차원을 줄이고, 10개 카테고리별로 스코어를 내는 공식을 배웁니다. 이것이 모든 딥러닝 뼈대의 시작점입니다.
- **Step 4: 여러 가지 관점 (대수/기하/시각)**: 선형 분류는 단순히 `W*x + b` 대수공식이기도 하며, 기하학적으로는 고차원 픽셀 공간을 가로지르는 분류 평면 선, 또 시각적 측면에서는 10장으로 겹쳐진 '사물 고유의 템플릿(형틀)'입니다.

## 2. 전문 용어 해설
- **L1/L2 Distance (거리 측정 공식)**: K-NN에서 두 사진 간의 픽셀 빼기 차이를 계산하는 방식. 절댓값 누적인 L1과, 유클리디언 제곱 평균 오차인 L2의 직관적 차이가 다뤄집니다.
- **Hyperparameter (초매개변수/하이퍼파라미터)**: 수학 모델 계산 중에 답을 찾는 가중치와 달리, '거리 측정 방식(L1인지 L2인지)', 'K의 갯수'처럼 사람이 학습 전에 "임의로 정해줘야 하는" 실험 변수들입니다.
- **Cross-Validation (교차 검증)**: 정해진 검증(Val) 세트로만 테스트하면 우연 요소가 개입되므로, 데이터의 조각을 `폴드(fold)`로 쪼개 서로 돌아가며 시험을 보게 해 안정성을 입증하는 기법.

## 3. 맥락 보완
- 과거의 컴퓨터 비전 규칙 기반(Rule-based) 알고리즘은 카메라가 사진 각도(Illumination), 스케일(Scale), 배경색 왜곡에 휘둘릴 때마다 모든 규칙 하드코딩을 다시 짜야 했으나, 데이터-주도 학습 선형 분류를 사용하면 수천 장의 라벨링 데이터가 템플릿이 되어 "왜곡을 통계적으로 포용"하는 능력을 갖추게 됩니다.
- 1시간 분량의 자막에서 집중적으로 설명되는 "기하학적 공간" 이야기는 선형 분류기가 오로지 "1개의 직선"으로만 판단하기에, 태극 문양(XOR 문제)처럼 꼬아져 있는 문제, 말마저 두 가지 색상이 섞인 특이한 공간은 죽었다 깨어나도 나누지 못하는 한계를 지적합니다.

## 4. 비유를 통한 설명
- **K-NN 시험생비유**: 평상시에 머리에 하나도 암기(가중치 구축)를 안 해놓고 있다가, 실전 문제가 종이로 주어지면 책상 위 1만 권의 백과사전을 모조리 펼쳐 하나씩 비교해가면서 대조하는 느려 터진 오픈북 응시자입니다.
- **선형 가중치(W)의 템플릿**: 각 클래스의 가중치 행렬 $W$의 개별 줄기(Row)는 '셀로판지 도장'입니다. 자동차 행렬이라는 셀로판 도장을 들고 이미지 위에 올려놓고 투영시켜 겹쳐지는 가장 진한 흔적이 스코어가 되는 이치입니다. 

## 5. 논문 소개
- **데이터 기반 접근의 원형 통계학/패턴인식 교재**: **Pattern Classification (Duda, Hart, Stork, 2000)** 같은 문헌에서 K-NN의 점진적 한계를 수학적으로 증명한 내용들이 이 강의의 기초 뿌리가 됩니다.
