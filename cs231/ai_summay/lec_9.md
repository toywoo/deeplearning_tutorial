# Lecture 9: Object Detection, Image Segmentation, Visualizing

## 1. 단계별 요약 (Step-by-step Summary)
- **Step 1: 검출 및 분할 태스크 (Det/Seg)**: 사진이 "고양이다"를 벗어나, 자율주행 차가 보행자와 차선, 사물의 위치마다 네모 박스(Bounding Box)를 치는 객체탐지 라인, 그리고 픽셀 단위로 마스킹 색깔을 따내는 의미론적 분할(Segmentation) 선을 긋습니다. 
- **Step 2: Region Proposal 과 Faster R-CNN 계열**: 영상 안에서 사물이 있을 만한 수 백개 위치 지역을 뜯어내 CNN을 무식하게 돌려 감지하는 클래식 딥러닝 디텍트 뼈대 구조입니다. 
- **Step 3: 단일 1-Stage 방식 (SSD, YOLO)**: 박스 후보군 추출 단계 수식을 폐기하고, 사진을 통째로 밀어넣어 한방 격자망 컨볼루션만에 Bounding Box와 점수 텐서를 동시다발로 즉시 토해내는 궁극의 실시간 추론 고속 모델입니다.
- **Step 4: 블랙박스 시각화 (XAI)**: 수천만 개 파라미터가 대체 어떻게 정답을 맞추는 지 신기하니, 가중치를 역전파해서 사물의 눈, 코 모양 등 어디서 활성화 점수가 요동치는지 Saliency 마스킹 열화상 검사로 원인을 시각화해 냅니다.

## 2. 전문 용어 해설
- **Semantic Segmentation (의미론적 분할)**: 사람 육안으로 사진의 픽셀 전 범위를 조사하여 나무는 초록 마스킹, 하늘은 파란색 통짜 마스킹 처리하는 클래스 분할 기술체계입니다.
- **IoU (Intersection Over Union, 겹침율)**: 모델이 예측한 테두리 박스 면적과 훈련 세트 정답 박스가 얼마나 겹치는지 오차 퍼센트를 따지는 교분 집합 수식 점수.
- **Saliency Map (관심 주의력 투영맵)**: 그 많은 사진의 배경색, 풍경 중에서도 네트워크 망이 가장 집요하게 의존했던 부위(중요한 픽셀)만 붉게 칠해주는 디버깅 X-ray 지도입니다. 

## 3. 맥락 보완
- 놀랍게도 2025 트렌드에서는 이 Object Detection도 박스를 뽑아내는 대신, 트랜스포머 계열 DETR 모델이 싹 다 갈아치우는 중입니다. NMS(중복박스 제거 알고리즘) 등 사람이 짠 휴리스틱 규칙도 다 필요 없이 트랜스포머의 어텐션이 박스마저 수학적으로 우아하게 뱉어내는 시대를 짚고 넘어갑니다.

## 4. 비유를 통한 설명
- **YOLO (You Only Look Once)**: 예전 R-CNN은 감식반이 사진 한장을 들고 "좌상단에 확대경 대보자... 다음 우측..." 하면서 지역별로 하나하나 검사했다면, YOLO 방식은 전지전능한 독수리 시야로 도화지 한번 쓱 전체 스캔에 눈을 번뜩이며 수십 명의 테러리스트 테두리에 1초 만에 마커칠을 해버리는 실시간 사격 통제망입니다.

## 5. 논문 소개
- **You Only Look Once: Unified, Real-Time Object Detection (Redmon 등 2016)**: 산업계 실시간 감시 카메라, 자율주행 시장의 폭주를 막을 수 없도록 만들어버린 세기의 속도 중시 디텍션 프레임워크 제안서입니다. 
- **End-to-End Object Detection with Transformers (Carion 등(DETR), 2020)**: 바운딩 박스를 뽑는데 사람이 만든 포스트프로세싱 제거 룰 따윈 필요없고 어텐션만으로 충분하다며 트랜스포머의 침공을 알린 기념비적 탐지 페이퍼입니다.
