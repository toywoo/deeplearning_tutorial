# Lecture 16: Vision and Language

## 1. 단계별 요약 (Step-by-step Summary)
- **Step 1: 멀티모달(대통합)의 개념**: "사진은 비전 네트워크, 글은 언어 네트워크"라는 파이프라인의 벽을 완전히 허물고 무너뜨려, 눈(시각)과 입(자연어 거대언어모델 LLM)이 한 우주 차원 벡터에서 교차하며 연관되게 엮는 인공지능 종착점 개요입니다.
- **Step 2: CLIP (대조적 텍스트-이미지 페어링)**: 인터넷에 널려있는 "강아지 밥먹는 사진 - [캡션: 귀여운 누렁이 식사]" 우주의 은하수 4억 개 쌍 데이터를 모아옵니다. 사진 인코더와 텍스트 인코더를 돌려 서로 맞는 짝꿍은 벡터 위상이 자석처럼 강하게 끌어당기게 대조 최적화를 가합니다.
- **Step 3: Zero-shot 추론 모델링**: CLIP 지식이 있으면 기존 컴퓨터 비전들의 고질병이었던 '지도 학습 클래스 수' 한계가 박살납니다. 그냥 수박 사진을 던져주고 "A photo of <수박>"이라는 무작위 자연어 문장 아무거나 1000개를 제시하면 그 중 내적이 가장 폭발하는 정답 문구 하나를 알아서 찾아 먹이는 압도적 제로샷 일반화. 
- **Step 4: LLaVA 모델 (VLM 등)**: ChatGPT-4 눈 달린 파트 등, 아예 대형 언어 두뇌(LLM) 에다가 CLIP 시각 인코더 시야 각을 브릿지로 우겨 쳐넣은 다이렉트 멀티모달 LLM. 사진을 주면 텍스트 스크립트로 썰을 푸는 만능형을 배웁니다. 

## 2. 전문 용어 해설
- **CLIP (Contrastive Language-Image Pre-Training)**: 텍스트와 이미지 간의 교집합 다리(Embedding Space)를 건설한 오픈AI 의 언어/비전 정렬(오리엔테이션) 기법.
- **Visual Question Answering (VQA)**: 모델에게 "저기 테이블 위 포크 옆에 무슨 과일이 있지?" 라고 이미지를 주며 질문을 던지면, 딥러닝이 저 답변인 "사과야"를 생성 텍스트로 내뱉어야 하는 비전 텍스트 융합형 테스트 포맷.
- **Zero-Shot Learning (제로샷 학습)**: 모델 설계자가 미리 라벨링 해서 가르쳐 준 적이 눈꼽만큼도 단 한 번도 없는 처음 보는 데이터 클래스가 등장해도, 이전에 배웠던 상식(글과 이미지 연관성)을 유추하여 정답을 한 방에 맞춰버리는 공포의 추론력.

## 3. 맥락 보완
- 현재 테크 기업(구글 제미나이, OpenAI GPT-4o 등) 의 생존경쟁에서 가장 주력 상품이 코어가 바로 이 멀티모달 VLM 입니다. 단순히 영상을 판별하는 공학을 넘어서, 로봇의 뇌수가 되는 인지 지능의 기반이 되므로 CS231n Spring 2025 학기 최고 중요 세션 중 하나입니다.

## 4. 비유를 통한 설명
- **CLIP 훈련의 벡터 대조 정렬**: 번역기를 쓰듯, '그림 우주 언어'를 통역해 주는 '텍스트 언어 사전 공간' 매핑입니다. 빨간 스포츠카 사진을 던지고 옆 방에 알파벳 글자 조각들을 흩뿌려두면, 이 모델이 영어 방에서 'C', 'A', 'R', 'R', 'E', 'D' 조각들이 자석처럼 끌려와서 차 그림에 막무가내로 찰싹 달라붙게 최적화 하는 통역 엔진입니다.

## 5. 논문 소개
- **Learning Transferable Visual Models From Natural Language Supervision (Radford, OpenAI, 2021)**: CLIP 이라는 단어를 이 세상에 알리며 멀티모달 일반화 신드롬을 폭발시킨 교과서.
- **Visual Instruction Tuning (Liu 등 (LLaVA), 2023)**: 비전 인코더와 대형 LLM 백본Vicuna을 조인트 브릿지로 결합시켜 미세조정(파인튜닝) 해내 오픈소스 멀티모달 LMM 모델의 가장 정석이 된 논문 설계.
